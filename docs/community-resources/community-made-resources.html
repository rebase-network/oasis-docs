社区工具

以下是一些社区创建的工具，可用于监测Oasis Testnet，自动执行节点部署等。

有关创建自己的工具，请看 Oasis Core 开发文档了解更多信息。

  • Oasisscan (开发者：bitcat)
  • Oasis Monitor (开发者：Everstake)
  • Oasis Hubble (开发者：Figment Networks)
  • Oasis.Fish (开发者：Stakefish)
  • Oasis Smartstake (开发者：Smartstake)

钱包, 图形化工具, 和抵押仪表板

  • Anthem (开发者：Chorus One)
  • Bitpie
  • RockX Wallet

  • Oasis Metadata Registry (by Oasis Protocol Foundation)
  • Oasis API Server (by Simply VC)
  • Oasis RPC Proxy (by Figment)
  • Oasis Indexer (by Figment)
