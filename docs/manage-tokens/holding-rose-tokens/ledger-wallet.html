Self-Custody With Ledger

使用Ledger管理钱包

Install Oasis App

要使用你的Ledger钱包来存放ROSE代币，请通过Ledger Live的管理器在你的Ledger钱包上安装Oasis 应用。

目前，你必须通过 设置->实验功能 启用 开发者模式，才能在Ledger Live的管理器中找到Oasis应用。

Oasis应用需要在你的Ledger钱包上安装最新的固件：

  • Nano S 的版本至少是1.6.1，在2020年8月11日发布。

Nano X 的版本至少是1.2.4-4，在2020年8月4日发布。

按照Ledger的说明更新你的Ledger钱包上的固件：

  • Nano S
  • Nano X

要在Ledger Live应用商店中找到Oasis应用，请进入应用目录的搜索栏，输入 Oasis：

要确保你至少安装了1.8.1版本的Oasis应用。

你可能会看到 Oasis 和 OasisVal 同时出现在搜索结果中。你要使用标准的 Oasis 。

安装后，Oasis应用程序将使用存储在Ledger钱包中的BIP 39助记词语为你的账户生成私钥和公钥对。
