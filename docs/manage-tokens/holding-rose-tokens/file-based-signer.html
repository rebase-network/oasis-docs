File-Based Signer

文件签名

我们强烈建议你不要在Mainnet上用文件签名生成的任何实体/抵押账户。

当使用文件签名时，强烈建议在离线状态使用。 获得对 entity、抵押账户的私钥的访问权限可能会使的你的 token 处于风险中。

你可以通过存储在文件中的私钥来持有 ROSE代币。
