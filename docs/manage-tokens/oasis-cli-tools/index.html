Oasis CLI Tools

Oasis 命令行工具

本节讲述如何通过 Oasis 命令行工具管理代币。

如果你不熟悉命令行，我们建议可以使用网页钱包。

Oasis 命令行工具仅支持 x86_64 Linux 系统。

x86_64 macOS 系统的支持还在开发中。当前，macOS 二进制包 releases of Oasis Core Ledger 包含 Oasis Core Ledger CLI and the Ledger Signer Plugin，但没有Oasis Node CLI macOS系统的二进制版本。如果想在 macOS 上使用Oasis Node CLI，需要从源码编译。
