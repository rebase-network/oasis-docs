Mobile Wallets

手机钱包

关于如何通过不同的移动钱包管理你的ROSE代币。

这些移动钱包还在开发中，所以不支持所有的代币相关操作。

如果你需要尚未支持的功能，请考虑使用Oasis CLI Tools。

Bitpie Wallet

  • 可用于 Android 和 iOS.

  • 目前支持地址生成和交易功能。

  • 要使用Bitpie钱包来管理你的ROSE代币，查看我们的Bitpie钱包指南。

Mainnet Beta禁止交易。

  • 正在开发抵押和代理功能。

Trust Wallet

  • ROSE代币信托钱包支持目前正在开发中。
  • 要了解更多关于Trust Wallet的信息，请访问网站。
